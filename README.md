# 🎲 Roll Dice Game

![Dice Game Thumbnail](./assets/thumbnail.png)

A modern **browser-based Dice Game** built with **HTML, CSS, and JavaScript**.
Players roll dice, compete for the highest score, and the results are stored in a **live leaderboard** using **Supabase database**.

This project demonstrates **frontend game logic, database integration, and real-time leaderboard functionality**.

# 🧠 How the Game Works

1️⃣ Two players enter their names
2️⃣ A target score is set
3️⃣ Players roll the dice turn by turn
4️⃣ Dice value gets added to their score
5️⃣ First player reaching the target wins
6️⃣ Winner and loser are saved to the **Supabase leaderboard**
